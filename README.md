## Project Context

This project was developed during the early stages of my Django learning journey.

It represents an important milestone in my growth as a backend developer and includes experiments with:

- Authentication & Authorization
- Role-Based Access Control
- Nested Comments
- Post Reactions
- Content Management

While the codebase does not fully reflect my current development standards, it showcases the concepts and challenges I explored at that stage.

My current primary project is WSS Platform, where these ideas are being redesigned and implemented with significantly more experience and architectural planning.
